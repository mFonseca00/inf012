# Fluxo Principal do Sistema Kos

Este fluxo resume as principais funcionalidades das páginas do sistema Kos, para apresentação rápida.

---

- [ ] **Acessar página de registro**
- [ ] Realizar registro como paciente
- [ ] Receber e-mail de confirmação
- [ ] Realizar login como admin
- [ ] Navegar pelo menu principal
    - [ ] Dashboard
    - [ ] Agendamentos
    - [ ] Listagem de pacientes
    - [ ] Listagem de médicos
    - [ ] Monitoramento
    - [ ] Criar novo paciente
    - [ ] Criar novo médico
    - [ ] Criar médico a partir do paciente do primeiro login    
- [ ] Agendar consulta como paciente
- [ ] Consulta aparece na lista do paciente
- [ ] Consulta aparece na agenda do médico
- [ ] Receber e-mail de confirmação de agendamento
- [ ] Cancelar consulta (paciente, médico ou admin)
- [ ] Receber e-mail de cancelamento
- [ ] ADMIN/MASTER acessa gestão de usuários
    - [ ] Listar usuários
    - [ ] Criar usuário
    - [ ] Editar usuário
    - [ ] Desativar usuário
    - [ ] Alterar perfil/permissão
